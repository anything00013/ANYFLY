Payment details (optional)
Airlines pay differently by country. Use whichever applies to you:

🇬🇧 UK — Sort code + Account number
Example: 60-16-13 / 31926819

🇪🇺 Europe — IBAN number
Example: GB29 NWBK 6016 1331 9268 19

🇮🇳 India — IFSC code + Account number
Example: HDFC0001234 / 50100123456789

🇺🇸 USA — Routing number + Account number
Example: 021000021 / 1234567890

🇦🇺 Australia — BSB code + Account number
Example: 062-000 / 12345678

🇨🇦 Canada — Transit + Institution + Account
Example: 00123-456 / 1234567890

🇵🇰 Pakistan / 🇧🇩 Bangladesh — IBAN
Example: PK36SCBL0000001123456702

⚠️ Security tip: Only share bank details directly with the airline after they confirm payment — not in your initial letter.
